# useReducer

`const [state, dispatch] = useReducer(reducer, initialArg, init?)`

Pozwala dodać reducer do komponentu.
- pozwala na przechowywanie i obsługę złożonego stanu w komponencie.
- jest swego rodzaju rozbudowanym `useState`
