# useSyncExternalStore

`const snapshot = useSyncExternalStore(subscribe, getSnapshot, getServerSnapshot?)`

Pozwala na subskrypcję do zewnętrznych zbiorów danych (store)
- subskrybuje do zewnętrznych store
- subskrybuje do API browserów
- wyciąga logikę do customowego hooka
- dodaje support do SSR (server side rendering)
