# useLayoutEffect

`useLayoutEffect(setup, dependencies?)`

Wersja `useEffect` która jest odpalana przed wyrysowaniem strony.
- ⚠️ wpływa negatywnie na performance
- pozwala na wyliczanie layoutu przed jego wyrysowaniem
